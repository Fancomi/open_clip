> 数据集：COCO val2014（Karpathy split），5000 张图像 CLS 特征  
> 运行命令：`bash analysis/probe.sh anisotropy`  
> 更新日期：2026-04-25

---

## 一、指标含义

### 1. Effective Rank（有效秩）

$$\text{EffRank} = \exp\!\left( H\!\left(\frac{\lambda_i}{\sum_j \lambda_j}\right) \right), \quad \in [1, D]$$

特征协方差矩阵特征值归一化后的 Shannon 熵指数化。值越大，说明特征在高维空间中越"均匀分散"——越各向同性；值越小，说明少数几个主方向主导了整个特征空间。

### 2. Participation Ratio（参与比率，PR）

$$\text{PR} = \frac{1}{D \cdot \sum_i \lambda_i^2 / (\sum_i \lambda_i)^2} \in (0, 1]$$

测量"有效参与"的维度占总维度的比例。PR = 1 代表完全各向同性，PR → 0 表示高度各向异性。与 EffRank 类似但对极端主方向更敏感。

### 3. Stable Rank（稳定秩）

$$\text{StableR} = \frac{\sum_i \lambda_i}{\lambda_{\max}} \in [1, D]$$

等价于归一化后的"不受最大特征值支配程度"。若稳定秩低，表示存在一个极端强势的主成分，特征分布极其不均衡。

### 4. Numerical Rank（数值秩）

$$\text{NumRank} = \#\{s_i : s_i \geq 1\% \cdot s_{\max}\}$$

奇异值中达到最大值 1% 以上的个数，即在数值上"不可忽略"的维度数量。代表特征的有效信息维度数。

### 5. Avg Cosine Similarity（平均余弦相似度）

$$\overline{\cos} = \frac{1}{\binom{N}{2}} \sum_{i < j} \cos(\mathbf{f}_i, \mathbf{f}_j)$$

随机 2000 个样本间的平均两两余弦相似度。值越高表示特征空间中大多数样本高度聚集（"collapse"趋势）；值趋近 0 表示分布接近球面均匀或中心对称。

> **方向**：↓ 越小越各向同性。

### 6. Std Cosine Similarity（余弦相似度标准差）

$$\sigma_{\cos} = \text{std}\left\{\cos(\mathbf{f}_i, \mathbf{f}_j)\right\}_{i < j}$$

两两余弦相似度的标准差。这是区分两种不同"各向异性"类型的关键指标：

- **Simplex / 多峰型**（如 DINOv3）：样本聚集在高维单纯形的顶点附近，不同类别间余弦差异极大 → **理论上 std_cos 应偏高**（COCO 5k 语义较集中，体现不明显；CC3M 上预计更显著）
- **低秩光滑流形型**（如 RADIO）：样本沿少数主轴聚集，同轴样本余弦相似度极高、不同轴样本差异大 → **std_cos 因轴内/轴间双峰分布而偏高**

> **注意**：std_cos 需结合 AvgCos 共同解读——RADIO 的 std 高来自低秩双峰，DINOv3 的 std 低来自 5k 样本语义集中。

### 7. pct_var_top{k}（前 k 主成分方差占比）

$$\text{top-}k\% = \frac{\sum_{i=1}^{k} \lambda_i}{\sum_i \lambda_i} \times 100$$

前 k 个主成分解释的方差百分比。若 `top4 > 40%`，说明特征几乎可以用 4 维表示，高度各向异性；若 `top4 < 10%`，则信息均匀分布在数百个维度，各向同性好。

## 三、逐模型深度分析

### 3.1 DINOv3 — 最各向同性，Simplex 结构

**核心数据**：EffRank=325.3（最高），PR=0.3833（最高），StableR=40.9（最高），AvgCos=0.0246（最低），top4%=7.8%（最低）

DINOv3 在所有"各向同性"指标上均排名第一。EffRank 达到 325，接近特征维度 768 的 42%；top4% 仅 7.8%，意味着前 4 个主成分只解释了不到 8% 的方差，信息均匀分散在数百个维度中。

**物理图像**：DINOv3 的自监督训练（无文本对齐约束、centering + sharpening）鼓励特征在语义上区分每个细粒度概念，特征空间形成高维单纯形（Simplex）结构——不同语义类的样本聚集在单纯形的不同顶点或棱上，顶点之间近乎正交，导致 AvgCos 接近 0，而整体 EffRank 极高。在 PCA 散点图中投影为"星形/凹陷多角形"轮廓。这不是平滑球面均匀分布，而是**多峰-顶点聚集型各向同性**。

**StdCos=0.0658 偏低的原因**：COCO 5000 样本本身语义分布相对集中，Simplex 不同顶点间的样本量差异大，平均后方差偏小。在 CC3M（100k 多样性样本）上重测，预期 StdCos 会显著上升。

**FPS 价值**：DINOv3 的 FPS 极值样本理论上落在单纯形棱末端，`--fps-model DINOv3` 选出的 5 个锚点在 PCA 图中应均匀覆盖边界区域，是跨模型视觉对比的优质锚点。

---

### 3.2 RADIO — 最各向异性，低秩压缩型

**核心数据**：EffRank=77.6（最低），PR=0.0281（最低），StableR=4.3（最低），AvgCos=0.5366（第二高），StdCos=0.1999（最高），top4%=40.1%（最高），top10%=50.6%（最高）

RADIO 是典型的**低秩光滑流形**结构。前 4 个主成分解释了 40% 的方差，前 10 个解释超过一半，整个特征空间高度压缩在少数几个主方向上。StableR=4.3 表示最大特征值主导极强。

**StdCos=0.1999（最高）的解释**：低秩结构意味着样本沿少数主轴排布，同轴样本余弦相似度极高（接近 1），跨轴样本差异大，导致余弦分布呈双峰或多峰，方差高。这与 Simplex 型的 StdCos 升高机制完全不同——RADIO 的 std 高源于**主轴聚集造成的轴内/轴间双峰**，而非真正的语义多峰性。

**如何理解**：RADIO 为多教师蒸馏模型（DINOv3 + SAM + SigLIP2），多个监督信号相互约束，特征在有限的"公共子空间"中聚合，损失了部分细粒度多样性，换来了跨域稳定性（COCO 与 CC3M 在 PCA 空间高度重合）和泛化能力。

---

### 3.3 SigLIP2 — 文本对齐带来的高聚集度

**核心数据**：EffRank=160.9，PR=0.1261，StableR=16.6，AvgCos=0.6443（最高），StdCos=0.0672，top4%=19.0%

AvgCos=0.6443 是所有模型中最高的——这是图文对比学习将图像特征向文本 token 方向压缩的直接体现。图像特征被"拉向"文本锚点，损失了图像内部的多样性，形成强烈的模态内聚集。但 EffRank=160.9（非最低）说明在这个聚集方向内部仍有合理的高维内部结构。

---

### 3.4 PE-Core — CLIP 系最均衡

**核心数据**：EffRank=203.5，PR=0.1685，StableR=21.4，AvgCos=0.3347，StdCos=0.0799，top4%=15.6%

PE-Core 在 CLIP 系（PE-Core、SigLIP2、TIPSv2）中各向同性最好，AvgCos 最低，说明图像特征的多样性保留相对最好。1024 维的更大特征空间可能有助于分散信息。

---

### 3.5 TIPSv2 — 接近 PE-Core，但更聚集

**核心数据**：EffRank=183.4，PR=0.1565，StableR=19.4，AvgCos=0.5313，top4%=16.0%

TIPSv2 与 PE-Core 各向同性指标接近，但 AvgCos 更高（0.53 vs 0.33），说明 TIPSv2 的图像特征更为聚集。使用更大分辨率输入（448×448）带来更多 patch 信息，但 CLS token 仍倾向于聚集在特定方向。

---

### 3.6 EUPE — DINOv3 与 RADIO 之间的混合体

**核心数据**：EffRank=125.3，PR=0.0888，StableR=12.2，AvgCos=0.1882，StdCos=0.1377，top4%=24.7%

EUPE 的指标处于中游。AvgCos=0.19 接近 DINOv3 的水平，说明特征方向较分散；但 EffRank 和 StableR 均明显低于 DINOv3，top4%=24.7% 远高于 DINOv3，说明存在一定低秩倾向。StdCos=0.1377 也较高，表明特征分布有多峰性——EUPE 同时具有 DINOv3 的各向同性和 RADIO 的低秩属性，可能源于其混合训练目标。

## 四、两种各向异性机制的本质区别

```
各向同性轴（EffRank）                          各向异性轴
◄──────────────────────────────────────────────────────►
DINOv3   PE-Core   TIPSv2   SigLIP2   EUPE   RADIO
 325.3    203.5    183.4    160.9    125.3    77.6

特征聚集度（AvgCos）
低 ◄────────────────────────────────────────────► 高
DINOv3   EUPE   PE-Core   RADIO   TIPSv2   SigLIP2
 0.025   0.188   0.335    0.537   0.531    0.644
```

| 特征 | DINOv3 型（各向同性-Simplex）| RADIO 型（各向异性-低秩）|
|------|------|------|
| EffRank | 极高（325） | 极低（77） |
| PR | 高（0.38） | 低（0.028） |
| StableR | 高（41） | 极低（4.3） |
| AvgCos | 极低（0.02） | 高（0.54） |
| StdCos | 低（0.066，受 5k 样本限制）| 高（0.200，低秩双峰） |
| top4% | 极低（7.8%） | 极高（40.1%） |
| PCA 视觉形状 | 星形 / 多角轮廓 | 椭球 / 香蕉形 |
| 产生原因 | 纯视觉自监督，细粒度语义区分 | 多教师蒸馏，跨模态公共子空间 |
| 下游优势 | 分割/检测 patch 特征丰富，FPS 极值有语义含义 | 跨域迁移稳定，低维可线性分类 |

**核心认知**：`EffRank / PR / AvgCos / top-k%` 均量的是"有多少维度被使用"，**无法区分"均匀球面"与"Simplex 多峰"**。`StdCos` 是识别 Simplex 结构的专用指标，但需要足够多样的样本（推荐 ≥ 50k）才能充分体现。

## 五、指标局限性与待改进方向

### 5.1 NumRank 饱和问题

所有模型 NumRank 均为 512（1% 阈值），说明 5000 样本协方差矩阵的有效秩已超过 512，无法区分模型。改进方案：
1. 阈值降至 0.1% 重测
2. 样本量增大至 50k+ 再测
3. 以 `stable_rank` 或 `participation_ratio` 为主要秩指标（不依赖样本量）

### 5.2 StdCos 对样本量敏感

DINOv3 的 Simplex 结构在 5000 样本下未充分暴露（StdCos 仅 0.066），建议在 CC3M（100k）上重测，预期 DINOv3 的 StdCos 会显著高于 RADIO。

### 5.3 COCO 语义偏差

COCO 是精选有标注数据集，语义分布偏向约 80 个类别。相比真实多样分布（如 CC3M），会低估各模型的多峰性强度，也会低估 RADIO 的域不变优势。

## 六、后续研究方向

### 方向 1：DINOv3 FPS 锚点验证 Simplex 结构

使用 `--fps-model DINOv3` 生成 `image_allmodels.png`，验证 FPS 选出的 5 个锚点是否在 DINOv3 PCA 图中均匀分布在"星形边界"，而在 RADIO PCA 图中是否聚集在主轴端点附近。若成立，FPS + 跨模型对比可作为结构差异的直观可视化工具。

### 方向 2：CC3M 上重测各向异性

预期：DINOv3 的 StdCos 显著上升（Simplex 顶点更多），RADIO 的 EffRank 变化有限（低秩对数据量不敏感），SigLIP2 的 AvgCos 略降。

### 方向 3：RADIO top-4 主成分的语义解释

RADIO top4 解释 40% 方差，这 4 个方向很可能对应跨域语义轴。实验：对每个 top PC，取高/低值端各 20 张图像可视化，验证是否对应可解释的语义维度（物体类别、场景类型、前景/背景…）。

### 方向 4：PCA 压缩后的 linear probe 性能曲线

DINOv3 top4=7.8%，RADIO top4=40.1%。问题：哪个模型在低维压缩后下游性能保留更好？

```
d in [4, 8, 16, 32, 64, 128, 256, full]
→ PCA 降维 → kNN or linear probe → COCO 检索 mAP / ImageNet top-1
预期：RADIO 在 d≤16 领先（信息集中），DINOv3 在 d≥64 追上（信息分散但总量大）
```

### 方向 5：训练过程中 Anisotropy 的演化

利用 epoch probe 链路观察：从 RADIO 初始化继续训练时，pct_var_top4 是否逐渐下降（向 DINOv3 靠近）？Simplex 星形轮廓形成于哪个 epoch？对应的 StdCos 变化轨迹如何？
